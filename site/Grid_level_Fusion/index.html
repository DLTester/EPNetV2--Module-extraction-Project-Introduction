Grid-level Fusion（网格级融合）¶

本阶段的作用是 将点级融合特征映射并融合到规则 BEV 网格空间中， 通过点到网格的聚合与 BEV 特征融合，构建统一的网格级多模态表示， 为后续基于 BEV 的卷积融合与检测提供输入特征。

本阶段包含三个核心算子：

  1. point_to_grid_scatter
    将点级融合特征按照点到网格的映射关系 scatter 到 BEV 网格中，完成点 → 网格的特征聚合。

  2. il_fusion_concat
    对来自不同模态或不同分支的 BEV 特征进行通道维拼接，形成融合后的网格特征。

  3. il_fusion_conv2d_fusion
    基于 2D 卷积的网格级融合网络，对拼接后的 BEV 特征进行空间上下文建模与融合增强。

API 文档¶

point_to_grid_scatter¶

point_to_grid_scatter(point_features, point_indices, grid_h, grid_w) ¶

IL-Fusion 模块中的点到栅格映射算子（Point-to-Grid Scatter Operator）。

将点级特征按照其在 BEV 网格中的索引位置， 累加映射到规则的二维 BEV 网格特征图中， 用于从 Point-level Fusion 过渡到 Grid-level Fusion。

Parameters:
  • point_features (Tensor) –

    点级特征张量，shape 为 (B, N, C)

  • point_indices (Tensor) –

    点在 BEV 网格中的索引，shape 为 (B, N, 2)， 分别表示 (y, x) 网格坐标

  • grid_h (int) –

    BEV 网格高度

  • grid_w (int) –

    BEV 网格宽度
Returns:
  • Tensor –

    torch.Tensor: BEV 网格特征张量，shape 为 (B, C, grid_h, grid_w)

il_fusion_concat¶

il_fusion_concat(tensors, dim=1) ¶

IL-Fusion 模块中的特征拼接算子（Concat Operator）。

将多个输入张量在指定维度上拼接，用于融合阶段整合点云与图像特征或多通道特征流。 拼接后的结果在输出前会被 clone，以避免内存别名问题。

Parameters:
  • tensors (List[Tensor]) –

    要拼接的特征张量列表，每个张量形状一致。 例如 [Fp_, Fi_gated]，均为 (B, D, N)。

  • dim (int, default: 1 ) –

    拼接维度（默认 1，对应通道维）。
Returns:
  • Tensor –

    torch.Tensor: 拼接后的融合特征张量。 形状根据拼接维度扩展，例如输入 (B, D, N) + (B, D, N) → 输出 (B, 2D, N)。

il_fusion_conv2d_fusion¶

ILFusionConv2DFusion ¶

Bases: Module

IL-Fusion 二维卷积融合模块（Conv2D Fusion Operator）。

该模块用于对经过 Point-to-Grid_level_Fusion 映射并完成特征拼接后的二维栅格特征 进行空间卷积建模，从而在规则网格空间中进一步融合点云特征与图像特征， 提取具有局部空间一致性的融合表示。

其输入通常来自多个点级融合特征在二维栅格上的聚合结果， 通过标准的二维卷积操作，在保持空间拓扑结构的前提下， 对不同模态信息进行联合建模与增强表达。

本模块属于 可学习算子，其参数在训练过程中根据检测或分割 等下游任务的监督信号进行更新。

结构说明（按顺序）¶
  1. Conv2d C_in → C_out, kernel=3, padding=1
  2. 在二维栅格空间中对融合特征进行局部感受野建模。
  3. BatchNorm2d
  4. 稳定特征分布，加速训练收敛。
  5. ReLU 激活
  6. 引入非线性表达能力，提升融合特征判别性。

（可根据需要堆叠多层 Conv2d，本实现给出最基础且通用的单层版本。）

输入¶

Args: grid_features (torch.Tensor): 二维栅格融合特征张量，shape 为 (B, C_in, H, W) - B：batch size - C_in：拼接后的融合特征通道数 - H：栅格高度 - W：栅格宽度

输出¶

Returns: torch.Tensor: 二维融合特征张量，shape 为 (B, C_out, H, W)

模块用途¶
  • 在规则二维空间中进一步融合点云与图像信息
  • 建模局部空间上下文关系，增强融合特征的空间一致性
  • 为后续检测头或其他下游网络提供高质量融合特征表示
  • 属于 IL-Fusion 的 Grid_level_Fusion-Level Fusion 阶段
